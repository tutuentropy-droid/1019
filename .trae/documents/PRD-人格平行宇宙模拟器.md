## 1. 产品概述

人格平行宇宙模拟器——用户输入一句话、一段经历或一个人物画像，系统基于心理学与社会学因果推演逻辑，生成此人在不同人生环境下可能演化出的3-5个平行人格版本。

- 核心目标：探索环境因素（家庭、时代、教育、创伤、资源）对人格结构的塑造路径，而非娱乐化的星座/MBTI测试
- 目标用户：对人格心理学、社会学、人生选择感兴趣的深度思考者，以及需要人物塑造灵感的创作者
- 产品价值：以因果推演替代刻板标签，让用户直观理解"如果人生不同，我会成为谁"

## 2. 核心功能

### 2.1 功能模块
1. **首页/输入页**：英雄区域、输入模块（三种输入模式切换）、推演触发
2. **推演加载页**：推演过程动画展示、因果因子可视化
3. **结果展示页**：3-5个平行人格卡片、深度对比视图、每个维度详情

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 首页 | 英雄标题区 | 产品标语、副标题说明、动态背景 |
| 首页 | 输入模式切换 | 一句话/一段经历/人物画像三种输入模式切换 |
| 首页 | 文本输入区 | 大尺寸文本框、字数统计、示例填充按钮 |
| 首页 | 推演参数控制 | 可选择平行人格数量（3-5个）、推演侧重点（家庭/时代/创伤等权重） |
| 首页 | 开始推演按钮 | 主CTA，带有加载状态和过渡动画 |
| 推演加载页 | 推演动画 | 展示从原始输入到人格裂变的过程，蝴蝶效应式动画 |
| 推演加载页 | 因果因子流 | 可视化展示家庭、教育、资源等因子如何作用于人格 |
| 结果展示页 | 平行人格总览 | 3-5个卡片横向/瀑布布局，每个卡片显示人格标签色和一句话简介 |
| 结果展示页 | 人格详情展开 | 点击卡片展开：性格描述、口头禅、价值观、人生选择、典型矛盾 |

| 结果展示页 | 因果溯源面板 | 展示每个人格版本的关键塑造事件链 |
| 结果展示页 | 重新推演 | 基于同一输入重新生成、或修改输入后再次推演 |

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深邃宇宙蓝（#0A0E1A）为背景基底，星云紫（#5B3FD9）、时空青（#00D4AA）为双强调色，人格暖金（#E8A838）为点缀
- **次色调**：石墨灰（#2A2F45）用于卡片，朦胧白（rgba(255,255,255,0.85)）用于正文
- **按钮风格**：圆角12px，主按钮使用渐变背景+微发光效果，悬停时有光晕扩散动画
- **字体**：标题使用「思源宋体」或类似衬线字体（营造学术与思辨感），正文使用「思源黑体」，数字/标签使用等宽字体
- **布局风格**：卡片式布局，叠加层设计，非对称栅格，大量留白与呼吸感
- **视觉元素**：星空粒子背景、流动的渐变光带、人格节点连接线、淡入淡出的墨迹晕染效果

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|---------|
| 首页 | 英雄标题区 | 衬线大标题、副标题、渐变分割线、背景流动星云 |
| 首页 | 输入区 | 分段切换Tab、大尺寸textarea、字数胶囊标签、示例快捷填充 |
| 首页 | 参数控制 | 滑块组件（人格数量）、因子权重标签组 |
| 首页 | CTA按钮 | 渐变背景、箭头图标、hover时光晕扩散 |
| 推演加载页 | 动画区 | 中心节点向外裂变、因子标签沿轨道流动、进度百分比 |
| 结果展示页 | 人格卡片 | 渐变顶部色条、人格代号标签、一句话描述、展开箭头 |
| 结果展示页 | 详情面板 | 五大维度标题+图标+正文、金句引号样式、时间线式因果溯源 |
| 结果展示页 | 雷达图 | 五轴极坐标、多色叠加填充、图例可交互切换 |

### 4.3 响应式
- 桌面端（>1024px）：人格卡片横向排列3-5列，详情面板侧滑展开
- 平板端（768-1024px）：卡片瀑布2列布局，详情面板底部弹出
- 移动端（<768px）：卡片单列纵向堆叠，全屏详情页，触控滑动切换人格
